Algoritmo de Kruskal

Algoritmo de Kruskal

Fue descubierto en 1956 por Joseph Kruskal, matemático y estadístico estadounidense.

Es un algoritmo de árbol de expansión mínimo, que toma un grafo como entrada y encuentra el subconjunto de las aristas de ese grafo que:

  • Forme un árbol que incluya todos los nodos iniciales.
  • Que tenga la suma mínima de pesos entre todos las aristas que se pueden formar a partir del gráfico.

Estructura Union Find

Kruskal hace uso de este algoritmo, ya que realiza dos importantes operaciones en una estructura de datos:

  • Operación Buscar
    Determina a cual subconjunto pertenece un elemento. Esta operación puede usarse para verificar si dos elementos están en el mismo conjunto.
  • Operación Unión
    Une dos conjuntos en uno solo.

Pasos

  1. Ordenar
    Se ordenan todas las aristas, con respecto a su peso de forma creciente.

  2. Agregar
    Se toma la arista con el menor peso y se agrega al árbol de expansión. Si al agregar la arista se crea un ciclo, prescindimos de esta.

  3. Repetir Se siguen agregando aristas hasta que todos los vértices estén conectados.

Pseudocódigo

    • Algoritmo kruskal(grafo):
      • conjunto_aristas={}
      • Ordenar por peso(grafo.aristas) ascendentemente
      • For each (u,v) ∈ grafo.aristas
        • Si Find(u) ≠ Find(v) Entonces
          • Union(u,v)
          • conjunto_aristas:= conjunto_aristas ∪ {(u, v)}
      • return conjunto_aristas;

Complejidad

  • Ordernar aristas
    Para el ordenamiento de las aristas.

    $$O(mlog(m))$$

  • unionFind
    Inversa de Ackerman

    $$O(a(n))$$

    Llamadas a unionFind()

    $$O(m*a(n))$$

  • Complejidad Kruskal

    $$O(mlog(m) + m*a(n))$$

      Selección y movimiento

      • Seleccionar nodo/arista

        Coloque el cursor sobre el elemento desea y hacer clic en él. Cambiará de color al estar seleccionado.

      • Mover nodo

        Seleccione el nodo y sin soltar el elemento, muévalo con el cursor a la posición deseada.

      • Mover grafo

        Coloque el cursor en una parte libre del plano y de clic. Sin soltar mueva el grafo al lugar deseado.

      • Zoom

        Para acercar o alejar el plano utilice el scroll de su dispositivo.

      Dibujar grafo

      • Agregar una nodo

        Para agregar un nuevo nodo debe dejar oprimida la tecla Ctrl en su teclado, seguido de esto al dar clic sobre el panel de dibujo.

      • Agregar una arista

        Una vez que tenga 2 o más nodos, debe seleccionar el nodo que desea unir. Posteriormente manteniendo oprimida la tecla Ctrl diríjase hacia el segundo nodo que desea unir y suelte el cursor sobre este.

      • Eliminar un nodo/arista

        Seleccione el nodo o arista que desea eliminar y oprima la tecla Supr para eliminarlo.
